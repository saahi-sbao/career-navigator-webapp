# **App Name**: Career Navigator

## Core Features:

- Career Field Presentation: Presents users with a dashboard of selectable career fields.
- User Authentication: Authenticate users to track career interests.
- Interest Tracking: Enables users to save their selected career field to their profile.
- Personalized Career Suggestions: Generate personalized career suggestions appropriate to the saved career interest.  The LLM uses a tool to determine the user's career interests.
- Real-Time Status Updates: Provides real-time updates on authentication and saving statuses.

## Style Guidelines:

- Primary color: Indigo-700 for main titles and key elements, conveying professionalism and trust.
- Background color: White for cards and #f4f7f9 for the body to ensure content readability.
- Accent color: Use theme colors (blue, green, red, yellow) for career cards to differentiate fields visually.
- Font: Inter sans-serif for a modern, readable interface. Use consistent font weights for hierarchy.
- Use clear, representative icons within career cards to quickly communicate the field.
- Card-based layout for career fields, ensuring a clean and intuitive user experience. Use a responsive grid for different screen sizes.
- Subtle hover effects on cards to provide feedback and enhance interactivity.
